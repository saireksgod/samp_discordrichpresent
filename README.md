# SAMP DISCORD REACH PRESENT
Плагин который отображает активность игры на любом SAMP/CRMP сервере в профиль Discord
За основу был взят плагин от Амира, но был добавлен cmake

## Building on Windows

```bash
cd samp_discordrichpresent
mkdir build && cd build
cmake .. -A Win32
cmake --build . --config Release
```
